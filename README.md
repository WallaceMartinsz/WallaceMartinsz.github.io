# 💻 Meu Portfólio Pessoal

Bem-vindo ao repositório do meu portfólio pessoal! Este é o meu espaço na web para apresentar quem eu sou, os projetos que desenvolvi, minhas habilidades técnicas e como você pode entrar em contato comigo.

Site : [`exemplo`](exemplo)

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando o ecossistema moderno do Next.js e React:

* **Next.js**
* **React**
* **Tailwind CSS** 
* **TypeScript**

## 🎨 Funcionalidades

* **Design Responsivo:** Adaptado perfeitamente para dispositivos móveis, tablets e desktops.
* **Modo Escuro/Claro (Dark Mode):** Transição suave de temas para maior conforto visual.
* **Seção de Projetos Dinâmica:** Exibição dos meus projetos.
* **Seção de Contato:** Exibição de meios para contato.

## 🛠️ Como Executar o Projeto Localmente

Siga os passos abaixo para rodar o projeto na sua máquina de desenvolvimento:

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/wallacemartinsz/portfolio.git
```

### 2️⃣ Entrar no diretório do projeto

```bash
cd portfolio
```

### 3️⃣ Instalar as dependências

```bash
npm install
```

### 4️⃣ Executar o servidor de desenvolvimento

```bash
npm run dev
```

### 5️⃣ Acessar o projeto

Abra [`http://localhost:3000`](http://localhost:3000) no seu navegador para ver o resultado.

**✅ Pronto!** Seu projeto está rodando localmente. Para parar o servidor, use `Ctrl + C` no terminal.
